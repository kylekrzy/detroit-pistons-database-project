# Detroit Pistons Database App

This project is a database application layer for the Detroit Pistons organization management system.

## Features
- Displays player, game, ticket, merchandise, and finance data
- Runs 5 MySQL analysis queries
- Connects to MySQL through PHP

## Technologies
- PHP
- HTML
- CSS
- MySQL
- MAMP

## How to Run
1. Start MAMP
2. Import `tables.sql`
3. Import `data.sql`
4. Place the project folder in `htdocs`
5. Visit `http://localhost:8888/detroit-pistons-database-project/app/`
